Programs of the BACI System

Program			Function
bacc			BACI C-- to PCODE Compiler
bapas			BACI Pascal to PCODE Compiler
bainterp		command-line PCODE Interpreter
bagui			Graphical user interface to the PCODE Interpreter (UNIX systems only)
badis			PCODE de-compiler
baar			PCODE archiver
bald			PCODE linker

This table comes from BACI System Separate Compilation Guide (November 14, 2001).
