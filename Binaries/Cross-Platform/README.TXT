BACI GUI contains a GUI frontend for bainterp. You can use this on Windows, since its folder doesn't include bainterp.

JavaBACI is a full Java implementation of the BACI system.

jBaci is a full Java implementation of the BACI system in an IDE.

BaciBeans refers to a NetBeans 7 plugin that allows compilation and interpretation of BACI.

NOTE: NetBeans 7 is not included. You will have to find an unofficial download as it is not offered anymore.
